# Local architecture notes (optional)

This folder is for **optional, machine-local** design notes during development
(architecture decision records, feasibility write-ups, milestone scratchpads).

Nothing under `docs/adr/` except this README is published to GitHub — see root
`.gitignore`. Shared, user-facing context belongs in `README.md`, `AGENTS.md`,
`docs/validation-checklist.md`, and `docs/testing/` instead.

If you keep ADRs here locally, treat them as personal reference only; agents and
contributors should not depend on them being present.
